## Hard-won DSL gotchas (full list in docs/spec/lowfat-filter-dsl.md → Authoring pitfalls)
`|` not comma for alternation · no flat-op-then-`match` in one body · `raw`-on-failure
defeats compaction when failure IS the bloat (mvn) · `or-shell:` runs on RAW · never
keyword-filter passthrough output · guard structured output (`--json`) byte-exact · guards
see only `exit ok|failed` · tail-anchored tools (pulumi): verdict at END, head/split cuts
the wrong half.
